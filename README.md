# SAP Deployment Factory

This template will

- Allow you to deploy SAP on Azure VMs
- you will need to edit the variables section of the template to add your subscription info

----
----

# Requirements:

**Infrastructure:**

- Azure Subscription
- Appropriate number of CPU Core quota
